    Biografía

    Bert Bos es un científico de la computación neerlandés, reconocido especialmente como de uno de los autores de la primera especificación de hojas de estilo en cascada.

    En 1995 Bert junto con Hakon Wium Lie hicieron una presentación conjunta en una conferencia de la World Wide Web sobre las hojas de estilo, donde Bert demostró la implementación de estas enArgo, un navegador en el que Bert había trabajado y que incluíaun sistema para aplicar hojas de estilo.

    En octubre de ese mismo año se une al W3C. A finales de ese año la W3C creó un grupo dedicado al desarrollo de HTML del que Bert formó parte. En 1996 publica junto con Harkon la primera especificación de CSS (hojas de estilo en cascada). En 1997 se unió como editor al grupo de trabajo para hojas de estilo. Este grupo se encargaría de las siguientes versiones de CSS. Bert es uno de los editores de las versiones 2 y 2.1 además de varios módulos de la versión 3.

    Datos adiccionales

    W3C

    También ha hecho aportes en HTML y XML dentro de W3C. Bert es considerado co-creador de CSS además de ser el representante del grupo de trabajo de CSS de la W3C.

    Sobre css

    CSS es el lenguaje de estilos utilizado para describir la presentación de documentos HTML o XML.
